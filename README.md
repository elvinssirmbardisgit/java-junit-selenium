# Java JUnit Project

A basic Java project with JUnit testing framework.

## Project Structure

```
src/
├── main/java/com/example/    - Source code
└── test/java/com/example/    - Test code
pom.xml                         - Maven configuration
```

## Building

Build the project using Maven:

```bash
mvn clean compile
```

## Running Tests

Run all tests:

```bash
mvn test
```

## Requirements

- Java 11 or later
- Maven 3.6+
